The beta is born

Mon Apr 30, 2018 by tercen

We worked hard to incorporate all the great feedback we gathered over the last year, now we are delivering a have a great beta version of the Tercen Cloud. We focussed on issues which researchers face day to day when dealing with large amount of data. We are very proud to deliver the following: Easy one button installation for New app store (i.e. library) on github Ability to import/export workflows New admin menu Improved user management New resource management (for CPU and Disk) Upgraded user interface New GUI access New data table views New progress bars New logs New scrolling modes Improved performance Github integration (for R code) New R code package management New full programmatic access to Tercen New Tercen R package, called teRcen and more… The beta version is available on request just send an an email to info@tercen. The feedback

Fri Sep 01, 2017 by tercen

After many sessions with researchers. We are delighted to sum up the major feedback we gathered during demonstrations of the alpha. Privacy of data is paramount especially when the researchers are working with measurement derived from human subjects The methodology approaches are just as important as the tool (i.e. knowing what to do with the tool as opposed what the features the tool offers) Open source of methodology code is non-negotiable for many of the researchers Stay tuned to see who we digest this and use it to put some cool features into the beta version. The alpha is born

Tue Jan 10, 2017 by tercen

After working throughout 2016. We have a great alpha version of the Tercen Cloud. We focussed on issues which researchers face day to day when dealing with large amount of data. We are very proud to deliver the following: Interactive multi-dimensional visuals Workflow editor A new visual for pairwise plots called multi-pairwise A wizard to help researchers perform workflows A rtercen R library Ability to organize researchers into Projects and Teams An up-to-date cloud-base architecture using docker, CouchDB and cephFS A invitation system for new users The alpha version allows us to demo the concepts and go out to the researchers and hear what they say about the approach.
